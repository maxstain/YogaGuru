## Features
- **Real-time feedback** on 4 yoga poses:
  - Goddess Pose 
  - Warrior II 
  - Downward Dog 
  - Tree Pose 
- **Biomechanical angle analysis** for precise corrections
- **Confidence scoring** (0.0-1.0)
- **interface** with camera feed
